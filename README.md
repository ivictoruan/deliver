### Prerequisites
Rode:
>> git clone url-deste-projeto
>> pip install -r requirements.txt


### Installing
Na pasta raiz (onde está o manage.py):
>> python manage.py makemigrations 
>> python manage.py migrate 
>> python manage.py runserver
